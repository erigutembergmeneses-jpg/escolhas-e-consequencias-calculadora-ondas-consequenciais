# 🌊 Calculadora de Ondas Consequenciais

Ferramenta interativa baseada em neurociência da decisão e economia comportamental para mapear o **efeito dominó** de escolhas profissionais, pessoais ou institucionais.

> *"Toda decisão desdobra-se em ondas. A primeira é direta e imediata. A segunda é indireta. A terceira é sistêmica. Seu cérebro enxerga a primeira com nitidez. Captura a segunda com esforço. Quase nunca registra a terceira. O problema é que a terceira onda é a mais cara."*  
> — Baseado em *Escolhas e Consequências*, Erigutemberg Meneses (2026)

## ✨ Funcionalidades
- 📊 **Mapeamento em 3 Camadas**: 1ª Onda (0-3m), 2ª Onda (3-12m), 3ª Onda (1-5a)
- 🔍 **Filtros de Validação**: Teste da Primeira Página, Teste da Repetição, Custo da Omissão
- 💾 **Persistência Local**: Salva rascunhos automaticamente no navegador (`localStorage`)
- 🖨️ **Exportação Limpa**: Gera relatório formatado para impressão/PDF
- 📱 **Responsivo**: Funciona em desktop, tablet e mobile
- ⚡ **Zero Dependências**: HTML, CSS e JS puros. Leve e rápido.

## 🚀 Como Usar
1. Clone ou baixe o repositório
2. Abra `index.html` em qualquer navegador moderno
3. Preencha os campos contextualizando sua decisão
4. Clique em **📊 Gerar Mapa** para visualizar o efeito dominó
5. Use **💾 Salvar Rascunho** ou **🖨️ Exportar/Imprimir** conforme necessidade

## 🌐 Deploy no GitHub Pages
1. Envie os arquivos para um repositório GitHub
2. Vá em `Settings > Pages`
3. Em `Source`, selecione `main` branch e `/ (root)`
4. Clique em `Save`. O link estará disponível em ~1 minuto.

## 📚 Referência Acadêmica & Teórica
Esta ferramenta operacionaliza os frameworks apresentados no livro:
- **CVDI** (Ciclo Virtuoso de Decisão Intencional)
- **Apêndice C**: Matriz de Consequências
- **Capítulos 8, 10 e 11**: Desconto Temporal Hiperbólico, Normalização do Desvio, 3ª Onda Invisível

> **Nota Metodológica**: Os modelos neurais (ACC, vmPFC, dlPFC, TPJ) e vieses são apresentados como modelos funcionais consolidados, não como mapas anatômicos literais. Ferramentas não substituem aconselhamento jurídico, clínico ou contábil.
